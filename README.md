# Cari Ayat

Landing page untuk aplikasi **Cari Ayat** — aplikasi untuk menemukan ayat Al-Qur’an dari lafaz latin atau kata kunci terjemahan bahasa Indonesia, dengan toleransi typo (fuzzy search).

## Setup

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Tech Stack

Astro, React, Tailwind CSS
